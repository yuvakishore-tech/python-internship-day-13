# Web Scraping Using BeautifulSoup

## Tools Used
- Python
- requests
- BeautifulSoup
- CSV
- VS Code

## Description
This project scrapes quotes from https://quotes.toscrape.com and stores them in a CSV file.

## Features
- Extracts quote text, author, and tags
- Handles missing HTML elements safely
- Saves data into a CSV file

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4
2. Run:
   python web_scraper.py

## Output
- quotes.csv containing scraped data
